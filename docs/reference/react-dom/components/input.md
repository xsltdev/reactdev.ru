### –ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –º–µ—Ç–∫–∏ –¥–ª—è –≤–≤–æ–¥–∞ {/_providing-a-label-for-an-input_/}

–ï—Å–ª–∏ –≤—ã –Ω–µ –º–æ–∂–µ—Ç–µ –≤–ª–æ–∂–∏—Ç—å `input` –≤ `label`, —Å–≤—è–∂–∏—Ç–µ –∏—Ö, –ø–µ—Ä–µ–¥–∞–≤ –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π ID –≤ `<input id>` –∏ [`<label htmlFor>`](https://developer.mozilla.org/docs/Web/API/HTMLLabelElement/htmlFor). –ß—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –∫–æ–Ω—Ñ–ª–∏–∫—Ç–æ–≤ –º–µ–∂–¥—É –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ —ç–∫–∑–µ–º–ø–ª—è—Ä–∞–º–∏ –æ–¥–Ω–æ–≥–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞, —Å–æ–∑–¥–∞–π—Ç–µ —Ç–∞–∫–æ–π ID —Å –ø–æ–º–æ—â—å—é [`useId`](useId.md).

### –ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –Ω–∞—á–∞–ª—å–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –¥–ª—è –≤—Ö–æ–¥–∞
